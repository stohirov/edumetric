# EduMetric CRM

Ta'lim muassasalari uchun o'quvchilar, davomat va natijalarni kuzatish tizimi.

## Texnologiyalar

- Next.js 16 (App Router)
- React 19, TypeScript
- Tailwind CSS 4
- Recharts, next-themes

## O'rnatish

```bash
npm install
npm run dev
```

Brauzerda [http://localhost:3000](http://localhost:3000) oching.
